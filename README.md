# 🚀 AI Powered Web App Builder (NxtBuild)

An intelligent full-stack web application that helps users **generate, manage, and build web apps using AI**. This project combines modern frontend technologies with a powerful backend to deliver a seamless AI-driven development experience.

---

## 🌟 Features

* 🤖 AI-powered app generation
* 🔐 User authentication (JWT-based)
* 📂 Create, edit, and manage projects
* ⚡ Fast and responsive UI using React + Vite
* 🌐 RESTful API backend (Node.js + Express)
* 🗄️ Database integration (MongoDB)
* 📦 Clean and scalable project structure

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

### 3️⃣ Setup Backend

```bash
cd server
npm install
node server.js
```

Backend runs on:

```
http://localhost:5000
```

---

## 🔐 Environment Variables

Create a `.env` file inside `server/` and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
